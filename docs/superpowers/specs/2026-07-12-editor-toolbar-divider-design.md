# 编辑器工具栏分隔线设计

## 背景

编辑器局部语法工具栏当前使用通用的浅色底边框，和正文区域之间的分界不够明显。

## 已确认方案

- 保持 `1px` 实线，不增加工具栏高度。
- 分隔线使用 `--workspace-panel-border`，与编辑器面板外框保持一致。
- 仅影响编辑器局部语法工具栏，不影响预览区、全局顶栏和其他抽屉。
- 为工具栏增加专属样式类，避免依赖通用 Tailwind 边框色。

## 验收标准

- 语法工具栏与编辑器正文之间存在清晰的横向分隔线。
- 分隔线为 `1px solid var(--workspace-panel-border)`。
- 现有工具栏布局、高度和滚动行为不变。
